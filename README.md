# Data Visualization on Heart Disease Dataset using Tableau

This repository contains my **completed Task 3** for the **Data Analytics Internship at CodeAlpha**.

In this project, I transformed raw healthcare data into meaningful and visually appealing charts and dashboards using **Tableau**. The purpose of this project was to make the data easier to understand, reveal useful patterns, and communicate insights through visual storytelling.

---

## 📌 Project Objective

The main goal of this project was to:

- Transform raw data into visual formats
- Create clear and meaningful charts
- Improve understanding through visual analysis
- Identify important patterns and trends in the data
- Present insights in a dashboard format
- Build a professional portfolio project using Tableau

---

## 📂 Dataset Used

- **Dataset Name:** Heart Disease Dataset
- **Source:** Kaggle
- **Link:** https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset

## ❓ Key Questions Explored

During this visualization project, I tried to answer questions such as:

- How many patients have heart disease and how many do not?
- Which age groups are most common in the dataset?
- Does heart disease vary across gender?
- Which chest pain categories appear more associated with heart disease?
- How are cholesterol levels distributed among patients?

---

## 📊 Visualizations Created

The following visualizations were created in this project:

### 1. Heart Disease Count
- Shows the number of patients with and without heart disease

### 2. Age Distribution
- Displays the age distribution of patients using grouped age ranges

### 3. Gender vs Heart Disease
- Compares heart disease occurrence across male and female patients

### 4. Chest Pain Type vs Heart Disease
- Shows how different chest pain categories relate to heart disease

### 5. Cholesterol Distribution
- Displays the distribution of cholesterol levels among patients

### 6. Final Dashboard
- Combines all visualizations into one interactive and clean dashboard

---

## 📈 Key Insights

Some important insights from this project include:

- The dataset shows a visible difference between patients with and without heart disease
- Most patients belong to a specific age range, showing age concentration in the dataset
- Heart disease occurrence appears to vary across gender categories
- Certain chest pain types appear more associated with heart disease than others
- Cholesterol values are distributed across multiple ranges, with some ranges appearing more common than others
